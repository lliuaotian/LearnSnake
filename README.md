## 测试
在Tests文件夹中有S./tests/Snake_test.cc文件可以用来测试功能
```c++
cd tests
g++ -g ../src/Input.cc ../src/Board.cc ../src/Snake.cc Snake_test.cc ../src/Food.cc -o ../build/snake
```
就可以在build文件夹中生成一个snake可执行文件，打开就可以运行了
